# Village Numérique Résistant — Web App (static)

Prototype multiplateforme en HTML/CSS/JS pour la Nuit de l'Info. Contient :

- Page d'accueil (index.html)
- Page actions (actions.html)
- Simulateur (simulateur.html)
- Page contribution (contribute.html)
- Styles modernes responsive et JS d'interaction

## Installation

1. Crée un dossier et les sous-dossiers `css`, `js`, `images`.
2. Sauvegarde les fichiers fournis ci-dessus dans les emplacements correspondants.
3. Lance un serveur local (recommandé) :

- `python3 -m http.server` puis ouvre `http://localhost:8000`.
- Ou utilise VS Code + Live Server.

## Déploiement

Dépose le dossier sur Netlify (drag & drop), GitHub Pages ou tout hébergement statique.
